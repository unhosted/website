Use your data anywhere, have it in one place.

The web is not as open as it used to be: big monopoly platforms have formed new proprietary layers on top of it. But we create a better architecture for the web. We break the “you get our app, we get your data” package deal with a cross-origin data storage protocol, separating application servers from data storage.

Unhosted is a non-profit project. To know more, check our mailing list and chat room. Our project is now one year old, see what we currently work on and where you can help.
